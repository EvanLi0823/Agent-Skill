# Projects 项目目录

存放AI Agent实践项目代码

## 目录结构建议

```
projects/
├── 01-basic/          # 初级项目
│   ├── simple-qa-agent/
│   ├── calculator-agent/
│   └── weather-agent/
├── 02-intermediate/   # 中级项目
│   ├── rag-system/
│   ├── multi-tool-agent/
│   └── memory-agent/
└── 03-advanced/       # 高级项目
    ├── multi-agent-system/
    ├── autonomous-planner/
    └── production-app/
```

## 项目模板

每个项目建议包含：
- `README.md` - 项目说明和运行指南
- `requirements.txt` - Python依赖
- `src/` - 源代码
- `tests/` - 测试代码
- `docs/` - 项目文档
- `.env.example` - 环境变量示例
